# Política de Privacidade

Este aplicativo foi desenvolvido com foco na simplicidade e na privacidade do usuário.

Coleta de informações

Este aplicativo não coleta, não armazena e não compartilha nenhuma informação pessoal do usuário.

Não solicitamos cadastro, login, e-mail, número de telefone ou qualquer outro dado que identifique o usuário.

Armazenamento de dados

Todas as informações inseridas no aplicativo (como itens da lista de compras e preços) são armazenadas exclusivamente no próprio dispositivo do usuário, utilizando armazenamento local.

Esses dados:
	•	Não são enviados para servidores externos
	•	Não são compartilhados com terceiros
	•	Não são acessados pelo desenvolvedor

Se o aplicativo for desinstalado, todos os dados armazenados localmente também serão removidos.

Integrações e serviços de terceiros

Este aplicativo não utiliza APIs externas, serviços de terceiros, ferramentas de análise, anúncios ou sistemas de rastreamento.

Segurança

Como os dados permanecem apenas no dispositivo do usuário, a segurança das informações depende das medidas de proteção do próprio sistema operacional do dispositivo.

Alterações nesta política

Esta Política de Privacidade pode ser atualizada futuramente. Caso isso ocorra, a versão mais recente estará sempre disponível nesta página.
